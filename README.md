# Prosjekt_IT_VG2

I dette prosjektet skal jeg lage en nettbutikk som inneholder ulike produkter som:
- Gamingutstyr
- Hardware
- Datautstyr (PC-komponenter og skjermer)

# Hvordan det skal se ut:
Det skal være en minimalistisk nettside med en meny fullt av ulike kategorier. Masse av gamingutstyr, det er hovedpunktet av nettbutikken.

# Planlegging:

Hva skal jeg lage de neste ukene?

- Skisser på figma
- Liten prototype
- Planlegge design på nettsiden
- Database (MariaDB eller MongoDB)
- Web-server (Bruk av Flask)

JEG SKAL LAGE BACKEND FØRST. FRONTEND KOMMER ETTER.

# Hvilket utstyr skal jeg bruke?

- Figma (Lage skisser)
- Visual Studio Code
   - Python Flask (Web-server)
   - HTML (Innhold til nettsiden)
   - CSS (Design av nettsiden)
- Database (MariaDB / MongoDB)

# Mitt prosjekt skal vise kompetanse innenfor IT-fagene (drift, utvikling og brukerstøtte)

- Drifting av nettsiden og alt av informasjon
   - Web-server (Vedlikehold av nettsiden)
   - Database (Informasjon av bestillinger fra kunder)
   - Feilsøking
  
- Utvikling
   - HTML
   - CSS
   - Python
   - Virtual environment (VENV)
   - Design
   - MariaDB / MongoDB

- Brukerstøtte
   - Brukerveiledninger
   - Dokumentasjon
   - Testing
